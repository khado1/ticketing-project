How to run the application


1. Open 2 terminals from the main folder (ticket-project)
2. 	Terminal 1: cd client
	Terminal 2: cd server
Make sure that you have yarn installed
3. Run yarn or npm install in both the terminals
4. Run yarn start or npm start in both the ternimals

In the server terminal you should see Nodemon starting the server.
In the client terminal you need to wait for the application ti build.
